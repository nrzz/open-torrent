# Security policy

## Supported versions

| Version | Supported |
|---------|-----------|
| `main` (unreleased) | Yes |
| Latest GitHub Release | Yes |
| Older tags | Best effort |

## Reporting a vulnerability

Please use **GitHub Security Advisories** on [nrzz/open-torrent](https://github.com/nrzz/open-torrent/security/advisories/new).

Maintainers ([@nrzz](https://github.com/nrzz), [@Dasprakash-Sekar](https://github.com/Dasprakash-Sekar)) triage private advisories. Do not open a public issue for remotely exploitable bugs until a fix is available.

Include:

- Affected platform (Android / Windows) and version/commit
- Steps to reproduce
- Impact (crash, RCE, data loss, etc.)

## Scope

OpenTorrent is a BitTorrent client. We do not police user content. Use the software lawfully. Issues about third-party copyrighted material are out of scope for this tracker.
