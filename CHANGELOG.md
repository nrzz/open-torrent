# Changelog

## 0.1.0 — initial public scaffold

- Flutter UI for Android + Windows (torrent list, details, settings, RSS)
- C API wrapper around libtorrent (`core/`) with stub engine fallback
- Session resume, sequential download, categories/tags, scheduler, SOCKS5 settings, IP blocklist path
- Android foreground download service + magnet/.torrent intent filters
- Windows tray / minimize-to-tray + Inno Setup installer script
- GitHub Actions CI + Release workflows
- Packaging stubs for winget and F-Droid
